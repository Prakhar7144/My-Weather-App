A responsive and dynamic Weather Application built using HTML, Bootstrap 5, and Vanilla JavaScript, integrated with the WeatherAPI service.

This project demonstrates real-time weather fetching, dynamic UI updates, geolocation detection, and clean modular JavaScript architecture.

Fetch real-time weather data for any searched city

# Features : 
Detect and display weather for current location (Geolocation API)

Default weather cards for predefined cities (Delhi, Kanpur, Pune)

Dynamic UI updates using reusable functions

Fully responsive Bootstrap layout

Clean, modular, and scalable JavaScript structure

Error handling for invalid cities

Separation of concerns (API logic & UI logic separated)

# Tech Stack
HTML5

Bootstrap 5

Vanilla JavaScript (ES6+)

WeatherAPI.com

Browser Geolocation API
